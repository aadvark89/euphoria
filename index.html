Methodology

How the precision analysis works

01

5-sec block scaling

Garman-Klass 5-minute realized volatility is scaled down to 5-second intervals using square-root-of-time law: σ₅ₛ = σ₅ₘ × √(5/300). This gives the true per-block vol aligned with Euphoria's bet timing.

02

Block-crossing probability

Given ETH price ~$2,500 and per-block σ, P(|Δp| ≥ $0.50) is computed assuming normal returns: P = 2 × Φ(−0.50/σ_dollar). This is the core metric for whether a 5-second bet stays within one price block.

03

2024–2025 ETF era adjustment

Post-ETF approval (July 2024) data shows measurably different microstructure. US institutional flow hours (13:00–16:00 UTC) became significantly more volatile. The 2024–2025 weighting is 60% of the composite vs 40% for 2018–2023.

04

30-minute window scoring

Each 30-min window is scored: micro-vol (35%) + block-cross probability (25%) + session activity (20%) + funding proximity (10%) + macro timing (10%). Windows scoring ≥75 are classified as prime tap trading conditions.
